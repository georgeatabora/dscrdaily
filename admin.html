Make.com Automation — Setup Guide
1Create a Make.com account at make.com (free plan works to start, $9/mo Core plan for scheduling)
    → Create a new Scenario
2Add a Schedule trigger — set to every day at 6:00 AM ET
    → This kicks off the scan automatically every morning
3Add an HTTP module → Make a Request
    → URL: your RentCast API endpoint for active listings
    → Headers: Authorization: Bearer [your RentCast key]
4Add a Filter module — only pass deals where DSCR ≥ 1.25
    → Formula: rent / pitia ≥ 1.25
5Add another HTTP module → POST to your webhook URL above
    → Body: the qualifying deal data (address, price, rent, DSCR, cash flow)
    → Admin panel receives it and queues for your review
6Add an Email module → send yourself a morning summary
    → "X new deals found today matching your DSCR criteria"
